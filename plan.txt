Prompt tem listas de prompts que são comandos que executam scripts e retornam o input do usuario



    - command
        - Pode executar um script
    - Script
            - tipos de script
                - executa ação

    - Prompt
        - Conjunto de palavras
            - executa um script

ou seja:

command - executa um commando (script)
    - prompt - script - é um script
        - prompt é um comando
